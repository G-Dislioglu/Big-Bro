# Contributing to Big-Bro

1. **Branch**: Create a feature branch from `main` (e.g., `feature/my-feature`)
2. **PR**: Open a pull request to `main` with a clear description
3. **Environment**: Copy `.env.example` to `.env` and set `ADMIN_KEY` and optional `DATABASE_URL`
4. **Development**: Run `npm install` then `npm run dev` for local development
5. **Testing**: Test your changes locally before submitting PR
